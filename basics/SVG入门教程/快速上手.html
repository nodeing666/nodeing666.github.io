# 1.快速上手

# 1.1.什么是svg？

SVG 是一种开放标准的矢量图形语言，即表示可缩放矢量图形（Scalable Vector Graphics）格式，是由万维网联盟（W3C）开发并进行维护的。

兼容性查询： https://caniuse.com/#search=svg

# 1.2.位图和矢量图的区别

位图：使用像素点来描述图像，也称为点阵图像。位图图像善于重现颜色的细微层次，能够制作出色彩和亮度变化丰富的图像，可逼真地再现这个世界，文件庞大，不能随意缩放；打印和输出的精度是有限的；

矢量图：矢量图使用直线和曲线来描述图形，这些图形的元素是一些点、线、矩形、多边形、圆和弧线等等，它们都是通过数学公式计算获得的。

# 1.4.svg的基本图形和基本属性

常用的图形

<rect>  矩形
<circle> 圆形
<ellipse> 椭圆
<line> 直线
<polyline> 折线
<polygon> 多边形

常用的属性

fill 填充颜色
stroke 描边颜色
strokewidth 描边粗细
transform 变形

# 1.画矩形

<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
  <rect
    x="10"
    y="10"
    rx="5"
    ry="5"
    width="100"
    height="100"
    fill="blue"
  />
 </svg>

矩形相关属性说明

x：矩形左上角x位置, 默认值为 0
y：矩形左上角y位置, 默认值为 0
width：矩形的宽度, 不能为负值否则报错, 0 值不绘制
height：矩形的高度, 不能为负值否则报错, 0 值不绘制
rx：圆角x方向半径, 不能为负值否则报错
ry：圆角y方向半径, 不能为负值否则报错

# 2.画圆形

 <svg xmlns="http://www.w3.org/2000/svg" version="1.1">
      <circle
        cx="100"
        cy="50"
        r="40"
        stroke="red"
        stroke-width="10"
        fill="orangered"
      />
    </svg>

圆形相关属性说明

cx 和 cy 属性定义圆点的 x 和 y 坐标。如果省略 cx和 cy，圆的中心会被设置为(0, 0)
r 属性定义圆的半径

# 3.画椭圆

<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
    <ellipse
      cx="100"
      cy="80"
      rx="90"
      ry="50"
      style="fill:green;stroke:purple;stroke-width:10"
    />
  </svg>

椭圆相关属性说明

cx 属性定义的椭圆中心的 x 坐标
cy 属性定义的椭圆中心的 y 坐标
rx 属性定义的水平半径
ry 属性定义的垂直半径

# 4.画直线

<svg xmlns="http://www.w3.org/2000/svg" version="1.1">
      <line
        x1="15"
        y1="15"
        x2="200"
        y2="200"
        style="stroke:green;stroke-width:2"
      />
    </svg>

直线相关属性说明

x1 属性定义了起始坐标的 x 值
y1 属性定义了起始坐标的 y 值
x2 属性定义了终点坐标的 x 值
y2 属性定义了终点坐标的 y 值

# 5.画折线

 <svg xmlns="http://www.w3.org/2000/svg" version="1.1">
      <polyline
        points="30,30 100,10 60,60 200,10"
        style="fill:none;stroke:red;stroke-width:3"
      />
    </svg>

折线相关属性说明

points, 点的集合，从第一个点依次连接后面的点，点和点之间用空格隔空，
点的x坐标和y坐标用逗号隔开 points="x1,y1 x2,y2 x3,y3"

# 6.画多边形

<svg xmlns="http://www.w3.org/2000/svg" version="1.1" style="height: 400px">
  <polygon
    points="200,10 250,190 160,210"
    style="stroke:purple;stroke-width:10"
  />
</svg>
