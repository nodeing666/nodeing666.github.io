# 2.1.Text组件

# 基本使用

Text(
  'hello world'*10, // 重复10次
  textDirection: TextDirection.rtl,
)

textDirection，设置文字放向,注意这个Text这个容器有一个宽度，里面内容也有一个宽度，如果内容宽度和Text的宽度一致，将看不出文字方向效果,例如：

如果将“hello world”重复10次，文本分两行显示，设置了文本方向即可看出效果，例如：

# 文本样式

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // TODO: implement build
    return Center(
      child: Text(
        'hello world' * 10,
        textDirection: TextDirection.rtl,
        style: TextStyle(
          fontSize: 50,
          fontWeight: FontWeight.bold,
          // color: Colors.blue,
          color: Color.fromRGBO(244, 0, 244, 0.3),
          height: 1.2,
          decoration: TextDecoration.underline,
          decorationStyle: TextDecorationStyle.solid,
          decorationColor: Colors.red,
          background: new Paint()..color = Colors.orange,
        ),
      ),
    );
  }
}


fontSize: 字体大小
fontWeight： 字体加粗
color： 字体颜色
height： 行高
decoration: 下划线
decorationStyle： 下划线样式
decorationColor： 下划线颜色
background: 文本背景

# 2.2.MaterialApp组件、 Scaffold组件

MaterialApp是一个封装好易于使用的Widget，它封装了应有程序实现Material Design所需要的一些Widget或者说是app开发中常用的符合Material Design设计理念的入口Widget。一般作为顶层(根组件)Widget使用，通俗的说MaterialApp是Material主题的一个根组件

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // TODO: implement build
    return MaterialApp(
      home: Home(),
    );
  }
}

class Home extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // TODO: implement build
    return Center(
      child: Text(
        'hello world' * 10,
        textDirection: TextDirection.rtl,
        // style: TextStyle(
        //   fontSize: 50,
        //   fontWeight: FontWeight.bold,
        //   // color: Colors.blue,
        //   color: Color.fromRGBO(244, 244, 244, 1),
        //   height: 1,
        //   decoration: TextDecoration.underline,
        //   decorationStyle: TextDecorationStyle.solid,
        //   decorationColor: Colors.white,
        //   background: new Paint()..color = Colors.orange,
        // ),
        textAlign: TextAlign.center,
        maxLines: 1,
        overflow: TextOverflow.ellipsis,
      ),
    );
    ;
  }
}

MaterialApp常用属性

title：这个和启动图标名字是不一样的，和当前 Activity 的名字也是不一样的。 这个 Title 是用来定义任务管理窗口界面所看到应用名字的。在原生 Android 系统中点击圆圈
theme：定义应用所使用的主题颜色，在纸墨设计中定义了 primaryColor、accentColor、hintColor 等颜色值。可以通过这个来指定一个 ThemeData 定义应用中每个控件的颜色。
color ： 定义系统中该应用的主要颜色
home ：这个是一个 Widget 对象，用来定义当前应用打开的时候，所显示的界面。
routes： 定义应用中页面跳转规则。 该对象是一个 Map<String, WidgetBuilder>。当使用 Navigator.pushNamed 来路由的时候，会在 routes 查找路由名字，然后使用 对应的 WidgetBuilder 来构造一个带有页面切换动画的 MaterialPageRoute。如果应用只有一个界面，则不用设置这个属性，使用 home 设置这个界面即可。如果 home 不为 null，当 routes 中包含 Navigator.defaultRouteName（'/'） 的时候会出错，两个都是 home 冲突了。如果所查找的路由在 routes 中不存在，则会通过 onGenerateRoute 来查找。指定默认显示的路由名字，默认值为 Window.defaultRouteName

Scaffold组件

Scaffold 是 Material Design 布局结构的基本实现。此类提供了用于显示 drawer、 snackbar 和底部 sheet 的 API。它是一个路由页的骨架，我们使用它可以很容易地拼装出一个完整的页面

        body: Home(),

      theme: ThemeData(primarySwatch: Colors.red),
    );
  }
}

class Home extends StatelessWidget {

    // TODO: implement build

AppBar是一个Material风格的导航栏，通过它可以设置导航栏标题、导航栏菜单、导航栏底部的Tab标题等。

# 2.3.Container组件

Container组件类似于html语言中的div，它的常用属性有：

width 宽度
height 高度
margin：外边距
padding：内边距
child： 子元素
transform：变换
decoration：背景装饰
alignment：对齐

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    // TODO: implement build
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('螺钉课堂')),
        body: Home(),
      ),
      theme: ThemeData(primarySwatch: Colors.red),
    );
  }
}

class Home extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Center(
      child: Container(
        // color: Colors.red,
        width: 400,
        height: 400,
        decoration: BoxDecoration(
          color: Colors.blue,
          border: Border.all(
            color: Colors.black,
            width: 10,
            style: BorderStyle.solid,
          ),
          borderRadius: BorderRadius.all(Radius.circular(40)),
        ),
        // margin: EdgeInsets.all(20),
        margin: EdgeInsets.fromLTRB(50, 80, 50, 40),
        padding: EdgeInsets.all(50),
        child: Text('hello nodeing'),
        alignment: Alignment.topRight,
      ),
    );
  }
}

}

  @override

        ),
        body: Home(),
      ),

    );
  }
}

class Home extends StatelessWidget {

  }
}
