# 3.String 字符串处理

# 3.1.什么是字符串对象？

使用单引号或者是双引号引起来的文本或者是通过new String([字符串])就是字符串对象.

var  str1 = “abcd”; //简单常用的定义形式
var  str2 = ‘abcd’;
var  str3 = new String(“abcd”); //标准定义形成
var  str4 = new String(‘abcd’);
var  str5 = ‘’ ; 也是字符串对象,只是字符串对象中没有字符.
var  str6 = “”;
以上都是字符串对象.

# 3.2.为什么使用字符串对象？

因为在JavaScript中为字符串对象提供了很多现成对字符串处理的方法和属性,借助于现有的方法和属性可以对对象中的字符串进行处理. 我们在工作中遇到的数据最多也是字符串类型的数据.

var str = "this is a big dog";
console.log(str.length);//17 
//length是字符串的属性，表示字符串的长度 通过str.length来使用

# 3.3.stringObject.charAt(index) 返回在指定位置的字符。

var str = "abcd";
console.log(str.charAt(0));//输出a
console.log(str.charAt(3));//输出3
console.log(str.charAt(3));//输出空字符串

总结：字符串中第一个字符的下标是0.如果参数 index 超出来字符串的长度，那么会返回一个空字符串，有时候需要拿到某个位置的字符串做判段的时候需要用到这个方法，比如去判定是id("#div")选择器还是类选择器(".div")的时候，就需要知道第一个字符到底是".",还是“＃”,这个时候就可以用charAt方法取出第一个字符去做判定

# 3.4.stringObject.indexOf(searchvalue,fromindex) 方法可返回某个指定的字符串值在字符串中首次出现的位置。

var str = "abcdefc";
console.log(str.indexOf("c"));//2 找到第一个出现的c的字符串的索引
console.log(str.indexOf("c",3));//6 从索引3开始计算 第一次出现字符串c的位置
console.log(str.indexOf("xxx"));//-1 在字符串中查询没有找到，那么就返回-1

总结：indexOf方法有两个参数，第一个参数是要搜索的字符串，第二个参数是从哪个位置开始搜索，它的执行过程就是拿着第一个参数里面的字符去字符串对象中搜索，看看到底有没有这个字符，有的话就返回这个字符的位置，没有的话就返回-1。通常可以配合字符串截取函数，在字符串中截取出想要的内容，比如截取文件后缀名、文件名等等，还要注意的细节就是这个方法对大小写敏感，而且是从前往后查找的

# 3.5.stringObject.lastIndexOf(searchvalue,fromindex)返回一个指定的字符串值最后出现的位置，在一个字符串中的指定位置从后向前搜索。

var str = "abcdefc";
console.log(str.lastIndexOf("c"));//从后往前搜索

总结：lastIndexOf方法和indexOf方法使用相同，只是搜索的顺序不一样而已，indexOf是从前往后搜索，lastIndexOf是从后往前搜索

# 3.6.stringObject.replace(substr,replacement)字符串中用一些字符替换另一些字符.

var str = "abcdefc";
console.log(str.replace("c","kkk"));//abkkkdefc
console.log(str.replace(/c/g,'kkk'));//abkkkdefkkk用模式字符串加上g替换全局,

总结:repalce方法有两个参数，第一个参数是需要被替换的字符串，第二个参数是被替换后的字符串，这个函数的执行过程要经过两步，第一步拿第一个参数去字符串对象中从前往后的查询，匹配上相同的字符串以后 第二步，用第二个参数的内容去替换第一个参数匹配上的内容，这里替换默认情况下只替换第一次查询出来的内容，如果要替换全局需要用正则表达式加全局g标记来替换

# 3.7.stringObject.substr(start,length) 可在字符串中抽取从 start 下标开始的指定数目的字符。

var str = "abcdefg";
console.log(str.substr(2,2)); //cd
console.log(str.substr(-2,2)); //fg

总结：substr方法用于字符串截取，有两个参数，第一个参数，是开始截取的位置，第二个参数是截取的长度，第一个参数如果为负数，那么位置从后往前计算，第二个参数可以不写，不写的话就从开始位置截取到字符串末尾

# 3.8.stringObject.substring(start,stop)用于提取字符串中介于两个指定下标之间的字符。

var str = "abcdefg";
console.log(str.substring(2,4)); //cd
console.log(str.substring(2)); //cdefg

总结：substring方法也是用于字符串截取的，和substr不同的是，substring方法截取的是两个位置之间的字符串，有两个参数，第一个参数是开始位置，第二个参数是结束位置，截取的就是中间的一段字符串，第二个参数可以不写，默认从开始位置截取到字符串末尾。

# 3.9.stringObject.toLowerCase() 把字符串转换为小写。

var str = "NODEING.COM";
console.log(str.toLowerCase());//nodeing.com

总结：toLowerCase方法比较简单，没有参数，只是字符串到一个大小写转换，把大写转换成小写

# 3.10.toUpperCase() 把字符串转换为大写。

var str = "nodeing.com";
console.log(str.toUpperCase());//NODEING.COM

字符串方法实例 点击上传图片按钮检测上传文件后缀是否支持指定的上传后缀 效果图如下：

  <html>
<script type="text/javascript">
	function checkFile(){
		var allowExt = ['jpg','png','gif','jpeg'];
		//>>1.找到html中的上传文件对象
		var file = document.getElementById('pic');
		//>>2.得到上传文件的值
		var value = file.value; //加入说 D:/xxx/yy/xxxx.jpg
		//>>3.获取文件的后缀名,先找到.的位置  
		var pos = value.lastIndexOf('.'); 
		//>>4.截取.后面的jpg,将后缀名统一小写,因为上传的后缀名可能有大写的
		var ext = value.substr(pos+1).toLowerCase();
		//>>5.检测后缀名是否在数组中
		var tag = false;
		var len = allowExt.length;
		for(var i = 0; i < len;i++){
			//使用数组中的每个元素和文件后缀做对比
			if(allowExt[i]==ext){
				//只要对比上说明文件后缀已经出现在数组中,将tag设置为true
				tag = true;
				break; //就没有必要继续循环了
			}
		}

		if(tag){
		   alert(ext+'后缀名支持上传');
		}else{
		   alert(ext+'后缀名不支持上传');
		}
		 

	}
</script>
</head>
<body>
<!--id在当前文档中代表唯一的表示-->
<input type="file" id='pic'/><br/>
<!--当点击上传按钮触发方法,将方法加到onclick上-->
<input type="submit" value='上传图片' onclick="checkFile()"/><br/>
</body>
