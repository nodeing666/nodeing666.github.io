# 4.Math对象 处理数学计算

Math对象是JavaScript内置的对象，用于处理数学方面的运算，Math的相关方法，可以直接拿出来用，Math对象的属性和方法属于必须掌握的内容

# 4.1.Math的属性PI

console.log(Math.PI);//3.141592653589793
//注意：PI 是 Math对象上的属性，要使用的时候必须通过"."的形式来使用，不能直接使用PI，必须挂载到Math身上

# 4.2.Math.abs() 返回一个数到绝对值

    var num = -10;
    console.log(Math.abs(num));//10
    var num = 10;
    console.log(Math.abs(num));//10
    //括号里面传入一个具体到数字 不要去传字符串或者其他乱七八糟的东西

# 4.3.Math.ceil() 对一个数进行向上取整

var num = 66.6;
console.log(Math.ceil(num));//67

var num = 77.2;
console.log(Math.ceil(num)) //78

var num = -88.2;
console.log(Math.ceil(num));//-88
//注意：向上取整的时候，如果里面的数是正数，那么就去掉小数点，然后整数加1  如果里面的值是小数的时候，直接去掉小数就是最后得到的值

# 4.4.Math.floor() 对一个数进行向下取整

var num = 77.2;
console.log(Math.floor(num)); //77
var num = 88.9;
console.log(Math.floor(num));//88
var num = -88.9;
console.log(Math.floor(num));//89

//注意:向下取整对时候，如果里面的数是正数，那么结果就是直接去掉小数点后对整数
如果里面的数是负数，那么就去掉小数点，然后再减一

# 4.5.Math.max() 求出n个数中的最大值

console.log(Math.max(1,4,4,345,2,45));//345
console.log(Math.max(34,234,56));//234
//注意：max里面可以传n个参数，能求出其中的最大值

# 4.6.Math.min() 求出n个数中的最小值

console.log(Math.min(1,4,4,345,2,45)) //1
console.log(34,234,56);//34
//注意：min和max相对 一个是求最小值 一个是求最大值，而且里面传的值都可以是多个

# 4.7.Math.pow(x,y) 求出x的y次方(幂)

console.log(Math.pow(2,2));//2的2次方 4
console.log(Math.pow(3,3));//3的3次方 27

# 4.10.获取指定范围内对随机数

//需求：获取10到30之间的随机整数
/**分析过程：
1、通过已经学过对知识，可以知道要生成一个随机数，需要用到Math.random()这个方法，能够获
取到0-1之间到随机数，包括0但是不包括1
2、如果把这个0-1到随机数扩大n倍，那么同样可以得到一个随机数，上面需求中是需要获得10到30
之间到随机数,那么我们可以得出，它们到随机变化范围就是20(最大值30减去最小值10)，也就是说
这个数到变化范围是在0到20之间变化，所以我们将0-1之间的随机数扩大20倍,也就是用
Math.random()*20,这个时候能得到0到20之间到随机数
3、由于0-1之间到随机数是没有包含1的 而我们需求中需要包含到最大值（30），所以我们到变化范
围也是需要取到最大到变化范围20，那么我们就可以在再将随机范围扩大1，
用Math.random()*(20+1),这个时候，能够取到20 但是取不到21，最大到时候可能取到20.9999，
然后我们通过向下取整就可以取到20了，Math.floor(Math.random()*(20+1))
4、从以上几步 我们能够得到10到20到变化范围了 最小到时候为0 最大到时候为20  那么，在次基
础上加上一个10，那么我们这个变化范围就变成了10到30.
*/

因此我们可以得出这样一个方法：
function getRandomInt(min,max){
    return Math.floor(Math.random()*(max-min+1)+min);
}
注意：其中max－min，也就是我们分析中到随机到变动范围，加1的目的是为了取到最大到随机数
