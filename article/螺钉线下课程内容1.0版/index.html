# 螺钉线下课程内容1.0版本

# 1.项目内容

自从我们这边发布了线下课程以来，很多同学都在问我学习的内容，基于项目保密，一直都没有对外公布，现在我们就来聊一聊螺钉的线下就业班到底学的是些什么内容

要聊具体内容之前，我们先简单的说一下市面上的线下培训课程的痛点，首先，从课程价格上来讲，学习费用+生活成本一般需要3万+才能搞定，其次，课程基于面试而非实战，这怎么理解呢？前段时间接到一个同学的咨询，参加过武汉某马线下双元教学培训的，下面是真实截图：

上面的同学并不是个列，这是目前市面上几乎所有培训机构都会遇到的问题，培训机构的存在最开始是具有很大意义的，为各大互联网公司输送了一大批的人才，但是随着培训机构的增多，每年产出的人员增多，培训的时间和招收学员的基础层次限制了培训出来也就是初级工程师的水平，因此，在目前看了市场上初级工程师处于饱和的状态的，也就是说越往后，培训机构越多，产出越多，竞争就会更激烈，怎么解决这么多学员的就业问题呢？因此，培训机构会专门设置一个就业部门，负责给参加培训的学员进行就业指导，说白了就是包装简历，教你怎么说谎忽悠面试官，从培训机构的角度来看，只需要将培训学生包装好，让学生成功忽悠了面试官，那么就算找到了工作，因此，真正的技术课程实际上是为了包装简历服务的，以前在我们机构，有时候会发现技术好的学生反而没有会忽悠的学生拿的薪资高，这让我产生了思考，这世界到底怎么了？

基于前面讲的，目前的线下培训机构大多数是基于面试的，而这个出发点本是好的，但是其中多了一条就是，这种面试是包装后的面试，这无形中增加了企业的成本，企业要招聘一个靠谱的工程师比较难，因此，就不得不把工作年限提高，随着企业把工作年限提高，培训机构又不得不把包装的年限做得更久，14年左右培训机构包装的工作经验是1年，现在已经要包装2年工作经验才能有机会获得面试资格，这种恶性循环下去，估计过几年企业都不招3年以下的了，随着包装年限的提升，需要的技术就会更多，react、vue、flutter、nodejs、小程序等等，这就出现了一种畸形的课程升级方案，因为成本的原因，培训机构不可能把培训时长无限延长，在有限的培训时间内，不停的增加内容，这就造成了很多参加线下培训的同学只了解技术名词，而没有真正的实战过，说白了就是培训几个月，也是没有开发经验的，举个例子：react先讲两天基础、再讲两天实战内容，写个小demo，这样就算学了react了，同样的技术，例如vue、angular也是这样的套路，即使这样，四个月要从零基础学到nodejs后端，掌握起来还是很仓促的

从各大机构的就业数据上来看，就业数据还是很好看的，但这些都是第一次统计但数据，你可以认为是成功忽悠了面试官进入公司的同学这批人的数据，而真正在公司能正常干下来，没有被开除或者自己离职这批人的数据，培训机构是不会统计的，这数据会相当难看，具上面截图那位同学说，他们班两个月以后有很多人已经放弃了，回去做了原来的工作，这样钱和时间不就都浪费了么

基于上面的痛点，我们螺钉课堂针对性的设计了课程内容，大环境我们是改变不了了，但是我们可以改变自己以适应当前的就业环境，目前的现象是要转行就业必须包装简历，这是是否能获得面试机会的第一步，如果你来螺钉课堂参加现下培训，我们也会一对一的给出包装简历的方案，但是和别家培训机构不同的是，我们包装的内容是要以扎实的项目经验为基础的，也就是说你可以通过包装拿到高薪，并且还要能在这个岗位上能干下来不被开除，这是需要有实际内容做支撑的

我们分析了市面上很多家培训机构以及我曾经服务过的培训机构的课程，大多数都是不连续的，不是以产品为中心的课程，说白了就是你学习了各种各样的技术，但是你并不能做出产品，这是比较蛋疼的，相当于任督二脉没法打通一样，实力没办法提升。我们希望改变这种现状，于是我找了我的设计师朋友和我一起先开发一套产品，其中，我作为产品经理先把需求、原型这些设计出来，然后我的设计师兄弟带领他的团队开始帮忙做各种效果，目前仅仅是pc端的页面就已经超过了200个了，我们还会把小程序、app打通，真正的做一个企业级的项目，我打算把这个项目作为长期维护的项目，功能需求升级、页面升级、代码版本升级这些都会在我们今后课程的维护中有所体现，如果你有了扎实的技术基础，加上真正的实战经验，找个功能还会难吗

我们这个项目是一个在线教育项目--螺钉学城，初级版本只会完成核心的功能，也就是用户从购买课程到观看学习课程这一核心业务要开发完毕，以下是大致的规划，我们线下班的第一期内容，只能完成下图中的核心部分，毕业后，找到工作了，有兴趣的同学可以和我们一起开发

要做完整的产品，其中的各种细节还是很重要的，如果你学习过我们的的线上课程，那么线下课程质量你也不用担心，螺钉这边做教育，除了赚钱我们还是有点情怀在里面的，全程都是干货，因此，我们才敢给你签订真实的不就业退全款的合同，相信在线下培训机构中，目前来说没有哪个机构敢这么做，包括那些已经上市了的大公司，相对于这些机构来说，我们都是很渺小的，谁会在意你交钱以后到底能不能找到工作呢？与其花钱去那些看上去很牛逼的大机构，不然来螺钉和我们一起成长...,交个朋友，多个圈子，积累点人脉也是很好的
