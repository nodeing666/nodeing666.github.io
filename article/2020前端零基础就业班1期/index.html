# 2.第二阶段，搞定Vue框架，提高生产效率

本阶段，主要是以Vue全家桶为主，我们讲从基础开始讲，结合大量的练习，让你深入框架原理，熟练使用框架，由于框架的更新非常的快，使用框架其实并不是你的核心竞争力，达到能使用框架的水平，只能说在找工作的时候，成功的几率会更大，在这个阶段学习完成后，找个4000左右的实习是完全没有问题的

由于框架更新太快，我们的课程也会跟着更新，谁也不知道2个月后的用法到底和现在是否一样，因此，详细的课程大纲，我们会在第二阶段开启的时候公布，以此类推，我们将在每个阶段开始上课之前，公布详细的课程大纲，需要做的项目

# 3.第三阶段，就业面试

这个阶段主要以面试题为主，帮助包装简历，进行面试指导，为什么我们要把面试相关的内容直接放到第三阶段呢？主要是考虑到很多同学迫切的想去找工作尝试，实际上自己出去面试也是一种实战，这其实能检测你的技术到底在市场上是什么样的价位，如果你能在这个阶段就能找到不错的实习，我们是非常建议你边工作边学习后面的内容，真正到工作中去提升技能

# 4.第四阶段，搞定React框架，涨薪利器

本阶段，主要是以React全家桶为主，咱们也是从基础讲起，从目前市面上的框架使用情况来看，Vue和React都是必学的，Vue的入门相对于React来说要简单一些，会React技术栈同等条件下可能会有更好的竞争力，但这不是绝对的，因此，大家在以后的技术积累的过程中，可以选择一个框架作为主力框架使用，另一个作为辅助，技术先精通再扩展会更好，这个阶段学完后，薪资至少5000+，妥妥的了

# 5.第五阶段，Node后端精讲，打通全栈

本阶段，主要是以Node后端为主，在这个阶段主要讲后端相关的知识，帮助你能打通前后端，具备全栈开发的能力，这样在以后的工作中和后端对接数据的时候也会游刃有余，提高工作效率，同时，如果这个阶段搞定了，你自己也可以利用自己业余时间搞点副业，接个单子小点的几千，稍微大点的几万到几十万不等，有很多人搞副业实际上比自己上班薪资还高

# 6.第六阶段，移动端开发

本阶段，主要讲Flutter做跨平台开发，微信小程序开发等，如果这个阶段搞定了就可以真正的具有多端开发能力了，PC端+H5端+APP端+后端这些你一个人都可以搞定，接个App开发的单子就是上万的收入，如果你有全端开发能力，能给别人一套完整的解决方案，挣得更多
