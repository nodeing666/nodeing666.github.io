# 线下保底保薪就业班

这些年随着互联网的发展，IT培训机构也如雨后春竹般涌现，由于进入门槛比较低，整个行业鱼龙混杂，质量差次不齐，培训体验更谈不上，性价比就更没法说了，经过我们多年的经验总结，我们以做互联网产品的思维，把用户体验与培训效果放在首位，打算做一款高质量、高性价比的IT培训产品，如何才能做出一款高质量、高性价比的培训产品呢？我们思考了以下几个方面的问题

1、传统培训模式有哪些坑是对用户极为不利的？

2、传统培训模式的效率如何才能被提升？

3、如何保证培训收益，减少转行风险？

4、如何最佳时期转行即转行的黄金期？

# 1.传统培训模式有哪些坑？

# 1.1.培训周期太长，时间成本增加？

传统线下培训一般都是零基础入门级别的培训，以前端开发为例，培训周期一般4.5-6个月，由于需要控制成本，一般都是4.5个月左右的培训时长，要是价格合理，培训时长6个月左右的机构就算是很良心的了，对于想要参加线下培训的朋友来说，长达半年的培训时长，这时间成本就非常的高了，这里说的时间成本是经济学上的机会成本，意思就是说你这半年如果没有参加培训，你去做别的工作一个月拿3000，半年就是1万8的机会成本，这里的成本是无形的，对于半年没有工作在城市里打拼的年轻人来说，生活压力也是非常大的，因此，如果要参加线下培训，首先要考虑的就是你到底有没有6-8个月的时间来脱产学习，自己的存款到底能不能支持自己无收入的情况下生活6-8个月（4-6个月培训，2个月找工作，保守得计划6-8个月时间）

# 1.2.培训人员基础不一致，合在一起上课效率低下？

通常报名线下培训班的同学，基础是不一样的，有的同学是工作几年转行来学习的，有的同学是刚毕业但是是别的专业的，比如英语专业的同学来学编程，有的同学可能是计算机专业毕业的，有几年的计算机基础，有的同学又可能是非计算机专业，但是自学过一段时间的，当这些基础不同的同学在一个班里上课的时候，该怎么讲课呢？线下培训班都是从零开始讲，都把所有人当成零基础的同学来讲了，这样一来，名义上是帮助有基础的同学系统入门，实际上是在浪费这些同学的时间和金钱来陪同那些零基础同学学习，这种现象非常普遍，也是非常不合理的，这些基础内容的学习至少要持续2个月-3个月左右，从这方面来说，时间都浪费来学基础了，整个培训效率肯定就不高了

这些都是在他们官网可以查询到的公开资料，平均学费都2万了，如果你不觉得高的话，说明家里有矿，可以随便去折腾了，除了学费以外，你还需要准备至少6-8个月的房租和生活费，以成都为例；单间算一个月700，吃饭一个月1500，车费一个月300，一个月生活最低得2500元， 那么你参加一次线下培训的实际支出成本计算为： 20000+2500*6 = 35000

这么高昂的费用支出，很多人是负担不起的，为了能让更多的人来培训，培训机构会和p2p金融公司合作，推出先就业后付款的模式，有很多同学会选择这种模式，但是潜在的风险却不是很清楚，这种操作模式说白了就是让你去金融公司贷款，由于金融公司和培训机构合作，你培训期间是不用还款的，但是一旦培训结束，不管你有没有找到工作，都是需要还款的，这个时候压力就来了，万一你找不到工作，利息加本金你需要还2万多的贷款，时间也浪费了，这个时候你去找培训机构退款，大概率是不会退的，潜在的转行风险高得可怕

# 1.4.传统培训机构培训后找不到工作的人多么？找不到工作怎么办？

传统培训机构出来找不到工作的人是有的，找到工作后胜任不了工作被开的情况也是有的，只是这些情况在培训机构的官方网站上都不会体现出来，你们能看到的各种培训机构的官网几乎都千篇一律的，高就业率、高薪资、还有些典型的同学被拉出来录视频说这个机构怎么怎么好之类的，当你非常迷茫需要一个梦想来支撑的时候，这些信息就是你最希望看到的，也是你最想要的，我们在这里想告诉你的是事物的另一面，你培训后很可能找不到工作，你面临的处境可能是继续干原来的工作，如果贷款了来培训的同学，你可能还面临着未来一到两年之内分期还款的压力

# 2.传统培训模式的效率如何才能被提升？

基于上面我们讲到的各种传统培训机构的坑，以及我们多年的探索，我们得出这样一个结论：

你完全可以只花一半的全职线下培训时间，一半的实际支出费用，几乎无风险的转行入门前端开发岗位

螺钉课堂线下培训班可以帮你实现最少的成本，最低的风险，转行入门前端开发岗位，入职薪资5000-12000，我们将从以下几个方面来改善传统培训机构的培训体验：

# 第一，只培训就业核心课程，缩短培训周期

我们已经开发了1500+课时的线上课程库，并且还在持续更新，如果想参加线下培训的同学，线上基础课程完全是可以免费自学的，那么就意味着线下培训班前2-3个月的基础培训课程被我们砍掉了，这样即节省了培训费用，又缩短了脱产学习的时间，这意味着你可以边工作边学习基础课程了，并不耽误工作，这样还有一个好处就是，在你自学基础的过程中，可以检验自己到底适不适合做这行，有很多人实际上是不适合做这行的，盲目参加培训把自己坑得很惨

那么，有同学会担心了，自学基础不牢靠怎么办？我们在线下讲项目就是综合利用前面基础课程里面的知识，在讲解的过程中会带着大家巩固知识的，实在担心在线下跟不上，可以自己多学一段时间再来也是可以的

# 第二，不提供金融贷款服务，学费可以分期付款

这里说的分期付款并不是和金融机构合作的分期付款，而是我们在你培训期间只收取一部分费用，找到工作后再收另一部分费用

# 第三，改善教学方式和教学环境以及课程质量

课程质量这块我们已经在网上验证了一年多，99%的同学对我们的课程质量还是比较认可的，但是这些线上课程只是作为我们线下课程的补充，我们线下将提供更具实战的项目，真实还原企业场景，同时教学方式上也会以少讲多练，培养编程思维和动手能力为主。这些都是硬性要求，除此之外，我们也会在其他细节上优化培训体验，例如；早上上课，我们可以错开早高峰，让大家不用挤地铁，中午吃饭，我们可以提前下课，让大家吃饭不用排队等等，我一直认为排队的时间、堵车的时间就是在浪费生命，耗费精力，让做事的专注力下降，还有更多的细节，我们一起来发现

# 3.如何保证培训收益，减少转行风险？

直接说解决方案：

1、入学签订保薪资保就业合同，保证就业

2、合同规定达不到就业薪资，全额退费

举个例子：

以成都为例：某同学参加培训，合同规定保底保薪薪资为：5000元，这位同学培训毕业后找到一个实习工作4500元，那么我们全额退费，一分钱不收取

# 4.如何最佳时期转行即转行的黄金期？

毕业后参加工作，时间就是寸土寸金，毕业3-5年对于个人职业生涯非常重要，如果规划得好，几年后在城市扎根，买车买房，成家立业相对来说更容易些，如果规划得不好，浑浑噩噩，混了5、6年后，觉得自己不适合这个工作再来转行，压力就越来越大了，我们这边一般都不建议年龄超过30岁的人转行做程序员了，试想一下，别人30岁已经有来8-10年的编码经验了，你才刚起步，这还怎么玩儿，公司需要的都是年轻力盛、头脑灵活、能加班能熬夜的年轻小伙，年龄大了技术还很小白，这样肯定是不好找工作的

根据我们的多年经验总结，转行做程序员，参加培训的黄金期是毕业后0-3年，这段时间转行做程序员是比较合适的，对于毕业3-5年转行做程序员的朋友成功率还是挺高的，说了这么多，总结一下就是，如果你是毕业0-5年，学历大专以上，可以考虑入行，并且不要犹豫不决浪费时间，如果你不满足这些基本条件，可以具体和我们聊了以后再做评估，千万不要盲目入坑
