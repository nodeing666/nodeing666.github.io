# 谷歌插件推荐

# 1.访问谷歌全家桶

对于程序员来说，使用谷歌搜索和安装谷歌浏览器的扩展插件会大大提高工作效率，有人说没有安装插件的谷歌浏览器只发挥了40%左右的功能，安装谷歌插件最快捷的方式是使用谷歌商店，但是由于某种原因，国内用户无法访问，这个时候就需要一个谷歌访问助手来解决，首先得能访问到谷歌商店，才能方便的安装各种插件

安装谷歌助手步骤：

第一步，下载这个插件，下载后解压到自己电脑的某个位置，注意：你需要记住这个解压的位置，并且保证这个位置不能带有中文路径，例如：“D：/软件/谷歌” 这种有中文的路径很有可能会报错，尽量不要解压到这种路径下

下载地址: 谷歌访问助手

第二步：打开谷歌扩展程序页面， 在浏览器输入 chrome://extensions/ 这个地址就可以打开了

第三步： 打开开发者模式

把上图这个按钮点开

第四步：把你下载的谷歌访问助手解压后的文件夹，加载到谷歌

经过以上步骤，即可使用谷歌的相关服务了

注意：这个插件可能会出现访问不了的不稳定情况，你可以重启插件等待一会儿再访问

# 2.免费广告屏蔽

点击进入谷歌商店

Adblock Plus - 免费的广告拦截器， 这个插件可以帮你免费屏蔽网站广告

# 3.一键翻译

Google 翻译插件可以帮助你翻译英文单词或者一键将英文网站翻译成中文

# 4.获取图片下载地址

Image Downloader 这个插件可以帮你获取当前网页中图片地址，可以方便的下载相关图片

# 5.美化json文件

JSON Viewer 这个插件可以将你在浏览器里面获取到的json格式的内容美化，方便查看

# 6.在浏览器中预览markdown文件

Markdown Viewer这个查看可以将md文件渲染成预览样式

# 7.美化标签页

Momentum 这个插件可以帮助美化标签页，美化后的标签页面是这样的

# 8.Octotree

这个插件可以以树形方式查看github中的开源代码

# 9.React Developer Tools

这个是react的调试工具

# 10.Redux DevTools

这个是redux的调试工具

# 11.Vue.js devtools

这个是vue开发调试工具

# 12.Tampermonkey

这个工具可以让浏览器运行一些脚本文件，例如：某云盘下载助手等，具体使用方法自行百度

# 13.书签侧边栏

这个可以很方便的管理书签，当你的书签存了很多的时候，你找起来比较麻烦，这个插件提供搜索功能，可以搜索你以前收藏的书签

# 14.捕捉网页截图

这个是一个很方便的网页截图工具
