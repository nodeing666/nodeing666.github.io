# 2020前端线上实战班1期-螺钉优品项目报名详情

# 1.项目由来

从我们录制前端课程开始，已经有数万人接触过我们的课程，有的同学学习了我们的课程成功找到了工作，有的同学已经从入门到放弃，还有的同学正在路上,不管你是什么情况，想要成功转行或者在职提升技术，我相信都需要一门更接近实战的课程，帮助你节省时间，提升效率，用技术挣更多的钱，过上不错的生活，在我们调研了市面上的几乎所有课程，我们发现一种现象，那就是几乎没有一家培训机构的课程，是完全基于一种产品进行迭代升级的，这是什么意思呢？举个例子：前端的知识非常庞杂，说白了就是要学习的内容非常多，各个机构的课程都是分阶段进行的，比如vue一个阶段、react一个阶段、nodejs一个阶段，实际上前期我们的课程体系也是根据知识来划分学习阶段，只是划分得更合理而已，但这依然不能满足我对前端课程体系的要求，按阶段划分课程体系其中的项目结合得并不是太好，比如，划分5个阶段，做了10个项目，但这些项目并没有什么难点，可以想象一下实际的工作场景，别人一个项目做了1年到2年，你自学或者培训5个月做了10个项目，数量虽然多，但是效果并没有预期的好，因此，我们螺钉课堂这边打算长期维护一个项目--螺钉优品，这是一个电商项目，我们希望通过这个电商项目，把前端需要用到的技术栈都串联起来，让你能形成非常系统的知识体系并具有真正的实战能力

# 2.为什么是电商项目？

实际上电商项目已经在网上泛滥了，但是各种电商项目都没有打通各个端，用到的技术也仅仅是一部分功能，比如使用React来写商城前端，但数据就是模拟但数据，后端功能就会缺失，我们希望做的是从头开始，前端、后端、App端、小程序端、桌面端都给打通，我们会尝试各种技术组合来完成一个庞大的电商系统，电商系统的深度和广度扩展性都是比较强的，我们会从核心功能开始，慢慢的迭代升级版本，课程也会随着版本的升级而推出，目前，第一季的内容，主要是把电商的核心流程给跑通，比如用户注册、商品管理、分类管理、订单管理、支付设置、购物车等等，从第二季开始就是重构、优化、扩展功能

# 2.这套课程是否适合我？

1.如果你是一个刚入门的新手，你已经学习了html、css、js基础，那么这套课程很适合你做进阶学习

2.如果你已经尝试学习了各种机构的课程，越学感觉越迷茫，以至于最后都不知道学什么了，那么这套课程很适合你，它将大大提升你的综合实战能力

3.如果你要想积累项目经验，有自己的作品去面试找工作，那么这套课程也很适合你，它是一个完全真实的项目，我们的计划是将这个项目完善后逐步开源，做一个Node技术栈的商城项目

4.如果你面临毕业，但是不知道毕业设计如何做，那么学这套课程也是非常合适的，它将帮助你顺利完成毕业设计，实际上很多大学的毕业设计是做不到这么复杂的，意味着你将它用于毕业设计是完全够用的

5.如果你是在职人员，想多掌握一门技术，做技术外包，接单挣钱，搞副业的话，那么这套课程也是非常适合的，商城项目应用到的知识非常的多，很多逻辑你甚至可以直接应用于别的项目，而你从零开始，从前端到后端，从PC到移动端都可以自己搞定的话，那么接一般的项目应该是没问题的

# 2.课程目标

1.熟练掌握Nodejs的核心模块，深入理解Js的运行机制

2.掌握Node相关的框架，提高生产效率

3.掌握数据库相关技术

4.掌握商城业务相关的开发技巧

5.最终目的是你要能独立的进行开发，举一反三，拿到需求就能自己独立的完成

# 5.上课方式及时间

1.线上直播，每周2-3次，每次 1个小时左右，手把手带着敲代码，分析实现思路
2.课后安排需求，每周末24点前提交本周作业
3.全程Git托管项目
4.课后配套一对一答疑
5.3月26号正式开班，根据课程的反馈以及接收程度安排下次课的直播时间，时间灵活
6.课后提供配套视频教程，作为作业辅助，没时间参加直播的同学可以课后学习我们的录播课程，为了提高学习体验，我们会单独录制知识点，并不是直播的随堂录制
